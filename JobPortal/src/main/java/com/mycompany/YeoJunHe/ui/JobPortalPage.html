Job Portal

    Search modes with examples:

    • Partial
      query engineer finds “Software Engineer”, “QA Engineer”, etc.
    • Exact
      query Software Engineer matches only that full title.
    • Multi-keyword
      separate terms with commas and mix title/location keywords with a salary floor. Example:
      • software,kl,3000 finds jobs whose title or location contains “software” AND “kl” (Kuala Lumpur) with salary ≥ 3000
    This is a local demo. Data is loaded in the browser.
